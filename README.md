# BilletePython

This is the Python version of the Billete application.

## Setup

1. Install Python 3.8+
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the main script:
```bash
python main.py
```

## Features (Planned)

- Flight data parsing
- PDF Document generation
- Word Document generation
- Layover time calculation
